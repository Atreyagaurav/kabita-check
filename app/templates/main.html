स्वागतम्


कविताको छन्द जाँच गर्नको लागि बनाइएको app /kabita_interactive मा गएर चलाउन सक्नुहुनेछ।

गलत शब्दहरु रातोमा देखाएको हुन्छ र त्यसमाथि mouse लैजाँदा के गलत छभनि देख्न सकिन्छ।

कविताको छन्द "I" र "S" ले छोटो र लामो स्वरलाई जनाउने गरी '#' पछि लेखिएको छ भने आफै जाँच गर्न सक्छ। उदाहरण /kabita_interactive को "Example" Button थिचेपछि देख्न सक्नुहुन्छ।

Welcome


Welcome to my page, this is just a placeholder, while the actual app is in /kabita_interactive endpoint. The API is available in /check endpoint, just send a JSON with {"text":"your kabita"}, it'll return the rendered HTML with mistakes highlighted in Red.
The mistakes are highlighted and explained in tooltip which you can see when you hover over them.
Save file and load file features are currently Unavailable.

The source code for this webapp as well as the backend if available in my github.
