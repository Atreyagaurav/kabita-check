कविताको छन्द जाँच गर्नको लागि बनाइएको app /kabita_interactive मा गएर चलाउन सक्नुहुनेछ।

कविता जाँच गर्ने र उदाहरण हेर्ने काम त्यहा गर्न सकिन्छ।

The testing web-app is in /kabita_interactive endpoint.

You can test your kabita as well as look at example kabita format there

कविता कसरी जाँच गर्ने

आफ्नो कविता unicode मा InputBox मा टाईप गर्न सक्नुहुनेछ; या आफ्नो फाईल बाट paste गर्न सक्नुहुनेछ। कविता तयार गरेपछि "Check" button थिच्नुहोस्।

कविताको छन्द "I" र "S" ले छोटो र लामो स्वरलाई जनाउने गरी '#' पछि लेखिएको छ भने आफै जाँच गर्न सक्छ। उदाहरण /kabita_interactive को "Example" Button थिचेपछि देख्न सक्नुहुन्छ।

To know how to check your kabita press the "Example" button and then look at the Example.

गल्ती कसरी हेर्ने

गलत शब्दहरु रातोमा देखाएको हुन्छ र त्यसमाथि mouse लैजाँदा के गलत छभनि देख्न सकिन्छ।

कति गल्ती भेटिए भनेर Analysis Results मा देखाउछ। रातोमा कुल गल्ती र हरियोमा कुल सहि देखाउछ।

The mistakes are highlighted and explained in tooltip which you can see when you hover over them.

Analysis result shows the overall analysis of the Kabita, red shows errors while green shows count of correct lines.

Informations for Programmers/Developers

  • The API is available in /check endpoint, just send a JSON with {"text":"your kabita"}, it'll return the analysis result as well as rendered HTML with mistakes highlighted in Red.
  • Save file and load file features are currently under development. I'm not great with javascript so the development is super slow.
  • I'd also like to integrate a nepali dictionary to check the correctness of words as well as chanda, and maybe suggest solutions for errors.
