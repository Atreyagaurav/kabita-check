Informations for Programmers/Developers

The API is available in /api endpoint, right now these are the only endpoints available, I'll add more as I develop them.
  • The API for checking chanda is available in /api/check endpoint.
    POST request; JSON: {"text":"your kabita","rule":"rule in is"}, it'll return the analysis result as well as rendered HTML with mistakes highlighted in Red. You can leave the rule empty for autodetection of chanda rule.
  • The API for chanda list is available in /api/all_chanda endpoint.
    GET Request.
  • The API for wordlist is available in /api/wordlist.
    GET Request; JSON: {"rule":"rule for word", "word":"word to search"}, the rule and word follow same syntax as that of query parameters on browse words page.
