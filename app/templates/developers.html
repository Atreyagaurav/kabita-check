Informations for Programmers/Developers

The API is available in /api endpoint, right now there are only two endpoints, I'll add more as I develop them.
  • The API for checking chanda is available in /api/check endpoint, just send a JSON with {"text":"your kabita"}, it'll return the analysis result as well as rendered HTML with mistakes highlighted in Red.
  • The API for chanda list is available in /api/all_chanda endpoint.

Other details:

  • Save file and load file features are currently under development. I'm not great with javascript so the development is super slow.
  • I'd also like to integrate a nepali dictionary to check the correctness of words as well as chanda, and maybe suggest solutions for errors.
